Sweep background images
=======================

Drop PNG files here named <TEAM>.png (uppercase 3-letter NHL abbreviation).
Examples: MTL.png, TBL.png, COL.png

These are used as the sweep background instead of the solid team primary
color. The diagonal sweep mask reveals the image as it animates across the
banner row.

Recommended dimensions: 700x120 (6:1 aspect ratio, native banner size).
Other aspect ratios will be center-cropped and scaled.

If a team has no file here, the animation falls back to the solid team
primary color (default behavior).
