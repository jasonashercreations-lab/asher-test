================================================================
  CSV DEFAULT: BANNER OUTLINE = TEAM PRIMARY (per side)
================================================================

In the CSV Default theme, the banner outline is now SPLIT:
  - LEFT half  = the away team's primary color
  - RIGHT half = the home team's primary color

So for ANA @ CBJ you'd see orange on the left, navy on the right.
For NYR @ BUF the split is invisible because both teams use blue.

Period/clock/stats grid stays plain WHITE as before.
Score row borders and penalty cell borders still use the CSV-curated
team accents.

Other 6 themes keep their unified single-color banner outlines:
  - Heritage   = warm brown across the whole banner row
  - Midnight   = dark blue-gray
  - Ice Rink   = light steel
  - Neon       = electric green
  - Stealth    = monochrome gray
  - Classic    = neutral gray (with team auto-swap on score/penalty)

Implementation: RenderColors gained two new fields,
away_banner_outline and home_banner_outline. CSV Default fills
them with the team primaries; every other theme fills them with
chrome (the existing single-color banner look).

================================================================
  HOW TO RE-UPLOAD
================================================================

1. Unzip somewhere temporary.
2. View -> Show -> Hidden items so .github folder is visible.
3. Drag everything inside the unzipped folder onto your local
   nhl-scoreboard folder. Replace all when asked.
4. GitHub Desktop:
     Summary: "csv default: split banner outline = team primary per side"
     Commit + Push
5. Wait ~10 min for the new .msi.
